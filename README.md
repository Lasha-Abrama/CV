# CV Light

CV Light is a modern, responsive online CV built with HTML and CSS. It showcases personal information, education, skills, and contact links in a clean and elegant layout.

## Features

Personal info & contact links

Education timeline

Skills with progress bars and circular indicators

## Technology

HTML5

CSS3 (flexbox, grid, gradients, animations)

## Usage

Open index.html in any modern browser. No server or installation required.
